About Gekko Browser

A lightweight, experimental web browser built with Electron

About

Gekko Browser is a modern, lightweight web browser built with Electron. It features a sleek, customizable interface and supports both standard web protocols (HTTP/HTTPS) and the experimental Gekko Protocol (GKP/GKPS).

Version: 1.0.0

Features

  • Modern, sleek interface inspired by Opera and Opera GX
  • Custom GKP and GKPS protocols for demo browsing
  • HTTP/HTTPS support for real web browsing
  • Tab-based browsing
  • Navigation history (back/forward)
  • Hyperlink support
  • Customizable themes
  • Built-in settings and history pages

GKP and GKPS Protocols

The Gekko Protocol (GKP) and its secure variant (GKPS) are specialized protocols for accessing resources across different domains.

Basic URL Structure

GKP URLs follow this format: gkp://<domain>.<tld>/

For secure connections, use GKPS: gkps://<domain>.<tld>/

Supported Top-Level Domains (TLDs)

  • .rust
  • .gekko
  • .kewl

Example URLs

  • gkp://home.gekko/ - Home page
  • gkp://about.gekko/ - About page
  • gkps://secure.gekko/ - Secure content demo

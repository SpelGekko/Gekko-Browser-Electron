About Gekko Browser

A lightweight, experimental web browser built with Electron

About

Gekko Browser is a modern, lightweight desktop browser built with Electron. It features a sleek, customizable interface and supports standard web protocols (HTTP/HTTPS) as well as internal demo protocols handled by the app: gkp:// and gkps://.

Version: 1.0.0

Features

  • Modern, sleek interface
  • Custom GKP and GKPS protocols for demo browsing
  • HTTP/HTTPS support for real web browsing
  • Tab-based browsing
  • Navigation history (back/forward)
  • Hyperlink support
  • Customizable themes
  • Built-in settings and history pages

GKP and GKPS Protocols

The app registers two internal stream protocols: gkp:// (demo protocol) and gkps:// (secure demo protocol). These are served from the packaged demo sites and app assets and are intended for local/demo content only.

How the app maps URLs

  • Special domains: shared.gekko (shared demo files) and assets.gekko (application assets like icons and fonts).
  • TLDs served: demo-only TLDs such as .rust, .gekko, and .kewl (only resources placed under src/demo_sites or the app assets/ directory are served).
  • Secure variant: gkps:// will add security headers (HSTS, CSP) and serves files from src/demo_sites/secure/ when present.

Example URLs (served by this app)

  • gkp://home.gekko/ — Local demo home page (from src/demo_sites/home.gekko).
  • gkp://about.gekko/ — This about page.
  • gkp://shared.gekko/theme-vars.css — Shared theme CSS served from src/demo_sites/shared.
  • gkp://assets.gekko/icons/32x32.png — App icon served from the packaged assets/ folder.
  • gkps://secure.gekko/ — Secure demo content (if present under src/demo_sites/secure/).
